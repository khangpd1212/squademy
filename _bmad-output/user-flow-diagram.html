03 — Quản lý phiên (Session Management)
🔐
Token-based Session
Access Token (JWT, 15 phút) — gửi trong Bearer header mỗi request.
Refresh Token (JWT, 7 ngày) — lưu trong localStorage, dùng để lấy access token mới khi hết hạn.
Cookie logged_in=true — đánh dấu trạng thái đăng nhập cho server component.

Khi access token hết hạn (401) → tự động refresh 1 lần → retry request.
POST /api/auth/refresh • POST /api/auth/logout • GET /api/auth/me
